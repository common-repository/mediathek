Mit diesem Plugin kannst du Videos aus den Mediatheken von ARD, ZDF und ARTE in Seiten und Artikeln anzeigen lassen.

== Description ==

*Dieses Plugin ermöglicht dir die Darstellung von Videos aus verschiedenen Mediatheken.*

Nachdem du dieses Plugin installiert hast, kannst du den folgenden Code in einen Artikel oder eine Seite einfügen:

[mediathek url="LINK"]

Achtung: Code nur über den HTML-Editor einfügen!
LINK sollte dabei durch den entsprechenden Link ersetzt werden (s. FAQ).

Eine kurze Anleitung zum Einbetten:

1. Rufe ein Video aus einer Mediathek auf.
2. Kopiere den Link.
3. Nun kannst du den Parameter 'url' eintragen.

<span style=\"text-decoration: underline;\">Die wichtigsten Funktionen im Überblick:</span>

*   Filme können auf der eigenen Seite betrachtet werden
*   Videos aus den Mediatheken von ARD, ZDF und ARTE werden unterstützt

Hinweis: Dies ist ein inoffizielles Plugin. Es wird keine Haftung für das Einbetten übernommen!

== Installation ==

1. Installiere das Plugin im Admin-Bereich oder extrahiere die ZIP-Datei, lade sie in den Ordner wp-content/plugins/ hoch und aktiviere das Plugin im Admin-Bereich.
2. Nun kannst du Artikel mit dem Shortcode [mediathek url="LINK"] erstellen.
